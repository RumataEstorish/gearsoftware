// FAQ

Common
questions.

Quick answers. If something's missing — ask in Telegram.

What devices are supported?
TaskEdge runs on Samsung Android phones with One UI and edge panel support. Tizen apps were built for older Samsung Galaxy Watch models running Tizen OS — that platform is discontinued.
Is the source code available?
All 10 Tizen apps are fully open source — browse, fork, learn from them freely. TaskEdge is a commercial product and is not open source.
How do I install TaskEdge?
TaskEdge is available on RuStore. It's temporarily unavailable on Google Play — check the product page for current status. Tizen apps are no longer in any store and require sideloading via the Tizen SDK.
Where do I get support?
The fastest way is Telegram — t.me/gearsoftware. Bug reports and feature requests are welcome there too.
Are the apps free?
All 10 Tizen open source apps are free. TaskEdge is a paid PRO product — check the product page for pricing. GearHub is a free Android companion app, but only useful alongside the Tizen apps it supports.
